# Design: A股全量信息检索知识库 — MVP原型

## 设计原则

1. **数据优先**：首屏最大化搜索框和数据展示区域，装饰元素最小化
2. **专业工具感**：配色和信息密度参考金融终端，但降低认知负荷
3. **时间线为核心**：事件链以垂直时间线展示，情感因子用色块直观标注
4. **单用户高效**：无需登录流程，Token在URL或localStorage中维护

## 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0f1117` | 深色背景，减少长时间使用疲劳 |
| 卡片背景 | `#1a1d2e` | 数据卡片，与背景形成层级 |
| 主文字 | `#e8eaf0` | 高对比度，适合暗色主题 |
| 次级文字 | `#8b8fa3` | 辅助信息、标签 |
| 强调色 | `#4c6ef5` | 链接、按钮、活跃状态 |
| 正面情感 | `#2ecc71` | 利好事件、正向因子 |
| 负面情感 | `#e74c3c` | 利空事件、负向因子 |
| 中性情感 | `#95a5a6` | 中性事件 |
| 警告 | `#f39c12` | 系统异常、ETL失败 |
| 健康 | `#2ecc71` | 组件运行正常 |

## 字体

- 主字体：系统默认无衬线字体栈（`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`）
- 代码/数据：等宽字体栈（`"SF Mono", "Fira Code", "Consolas", monospace`）

## 屏幕覆盖

| 屏幕 | 对应旅程 | 原型实现 |
|------|----------|----------|
| S1 搜索仪表盘 | J1-1, J2-1 | 首页Tab，大搜索框 + 最近查询快捷入口 |
| S2 个股事件链 | J1-2 ~ J1-5 | "事件链"Tab，垂直时间线 + 类型过滤 |
| S3 搜索结果 | J2-2 ~ J2-5 | "搜索"Tab，结果列表 + 关键词高亮 |
| S4 因子数据 | J3-1 ~ J3-4 | "因子"Tab，表格 + 简单SVG图表 |
| S5 系统状态 | J4-1 ~ J4-4 | "系统"Tab，健康指示灯 + ETL日志 |
| S6 事件详情 | J1-3, J2-3 | 模态弹窗，点击事件/结果触发 |

## 交互说明

- 顶部导航栏切换主屏幕（搜索/事件链/因子/系统）
- 搜索框支持股票代码和关键词两种模式
- 事件链支持按类型过滤（点击标签切换）
- 点击事件条目弹出详情面板
- 因子页面支持表格/图表切换
- 所有交互使用JavaScript实现页面内状态切换，无真实后端
